# Producer Pal API Reference

Краткое описание основных инструментов Producer Pal для работы с Ableton Live.

## Tools

### ppal-read-live-set

Читает информацию о текущем Live Set (проекте Ableton Live).

**Возвращает:**
- Структуру проекта
- Список треков
- Общие параметры проекта

---

### ppal-read-track

Читает информацию о конкретном треке в Live Set.

**Параметры:**
- `track_id` или `track_name` - идентификатор или имя трека

**Возвращает:**
- Параметры трека
- Список клипов на треке
- Настройки канала

---

### ppal-create-clip

Создает новый клип на указанном треке.

**Параметры:**
- `track_id` или `track_name` - целевой трек
- `clip_name` - имя клипа
- `notes` - массив нот (см. формат Note ниже)

**Возвращает:**
- Информацию о созданном клипе

---

### ppal-read-clip

Читает информацию о клипе, включая его ноты.

**Параметры:**
- `track_id` или `track_name` - трек, содержащий клип
- `clip_id` или `clip_name` - идентификатор или имя клипа

**Возвращает:**
- Параметры клипа
- Массив нот в формате Note (см. ниже)

---

## Формат Note

Каждая нота представлена объектом со следующими полями:

```python
{
    "pitch": int,      # MIDI pitch (0-127)
    "start": float,    # Время начала в тактах/битах
    "duration": float, # Длительность в тактах/битах
    "velocity": int    # Velocity (0-127)
}
```

**Пример:**
```python
note = {
    "pitch": 60,      # C4
    "start": 0.0,     # Начало на первом бите
    "duration": 1.0,  # Длительность 1 такт
    "velocity": 100   # Средняя громкость
}
```

---

## Примечания

- Все временные значения указываются в тактах/битах относительно начала клипа
- Pitch соответствует стандарту MIDI (60 = C4, 69 = A4)
- Velocity определяет громкость/силу нажатия (0-127)
